1. Clone this repo:
```sh 
git clone https://github.com/flubyGit/ui-clone-githubdesktop.git
```

## Go to directory

```
 cd ui-clone-githubdesktop
```

## Install dependencies

```
 yarn install
```

## Run server

```
 yarn start
```
